# PracticePromiseKit

This project demonstrates how to use PromiseKit that is a iOS library.

## Example

To run the example project, run `pod install` from the Example directory first.

## Keynote

See the PromiseKit.key file.

## Requirements

- Xcode 9+
- Swift 4+
- CocoaPods 1.3.1+
